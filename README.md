# Pagina de catalogo de productos (React + Vite)
Proyecto frontend desarrollado para una prueba tecnica para Pierce Commerce.


---
## Requisitos

- **Node.js** instalado
- **npm** (viene con Node.js)
- Conexion a internet para consultar las APIs externas

---

## Instalacion

1. Clona el repositorio:
   - `git clone <URL_DEL_REPO>`
2. Entra a la carpeta del proyecto:
   - `cd <NOMBRE_DE_LA_CARPETA>`
3. Instala las dependencias:
   - `npm install`
4. Edita el nombre del archivo .env.example a .env

---

## Ejecutar en desarrollo

- renombra  `.env.example` a `.env`

- `npm run dev`

Luego abre la URL que aparece en la terminal
